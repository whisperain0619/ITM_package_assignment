# ITM_package_assignment
simple python module

#package structure

myutils_package/

├── init.py # 패키지 초기화 파일
├── calculator.py # 계산 관련 함수 (add, multiply, power)
├── text_processor.py # 문자열 처리 함수 (reverse_text, count_words)
├── data_analyzer.py # 데이터 분석 함수 (calculate_mean, find_max, find_min)
└── examples/
└── demo.py # 예시 데모 파


# main fuction

### 1. Calculator Module (`calculator.py`)
- `add(a, b)`: 두 수를 더합니다
- `multiply(a, b)`: 두 수를 곱합니다
- `power(base, exponent)`: 거듭제곱 계산

### 2. Text Processor Module (`text_processor.py`)
- `reverse_text(text)`: 문자열 뒤집기
- `count_words(text)`: 단어 개수 세기
- `remove_spaces(text)`: 공백 제거

### 3. Data Analyzer Module (`data_analyzer.py`)
- `calculate_mean(numbers)`: 평균 계산
- `find_max(numbers)`: 최댓값 찾기
- `find_min(numbers)`: 최솟값 찾기
